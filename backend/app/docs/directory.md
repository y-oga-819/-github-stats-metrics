# ディレクトリ構成

```
backend
├── app
│   ├── application // UseCase
│   ├── cli // コマンドライン操作用ツール
│   ├── cmd // アプリケーションのスタートライン
│   ├── config // 各種設定値
│   ├── docs // APIドキュメント
│   ├── domain // ドメインオブジェクト
│   ├── infrastructure // DBや外部APIの実装
│   ├── presentation // ユーザーへの表示/入力
│   └── server // HTTPサーバーやルーティングの設定
└── ops // 各種環境周りのファイル
```
